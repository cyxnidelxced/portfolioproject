Web Design

Advanced Layout

Project Overview

This project is made to play with creative and unconventional layout ideas while continuing to learn how to use CSS Grid. Boring layouts were not allowed, and posters, print design, motion graphics, fine art, architecture, etc. can be used for inspiration.

My Role

Design a page using advanced CSS Grid

Timeline

February 27, 2025

The Solution

The Joker-inspired web design explores the creative potential of CSS Grid by creating a dynamic, non-traditional layout that breaks conventional design rules. By leveraging advanced CSS techniques, I developed an immersive visual narrative that captures the chaotic essence of the character.

  • Implemented a complex CSS Grid layout using grid-auto-flow and custom grid-area configurations to create an unconventional, asymmetrical design
  • Developed a responsive grid system that adapts to different screen sizes while maintaining the original design's creative integrity
  • Created interactive image hover effects using JavaScript to add depth and dynamism to the page
  • Utilized creative positioning and overlapping grid areas to generate visual interest and narrative flow
  • Incorporated custom typography and a distinctive color palette to enhance the thematic experience
